# Troubleshooting

## Convex URL Is Missing

If the app cannot connect to Convex, verify `NEXT_PUBLIC_CONVEX_URL` in `apps/web/client/.env.local` and restart the app process.

## Runtime Cannot Read Project Files

Project files must live under `.onlook/sandboxes/<project-id>`. If file reads or writes fail, verify that the folder exists and that the local runtime is using the expected project id.

## Preview Is Stale

Use the runtime restart control in the editor. The preview is served by the project's local dev process, so stale output usually means the process needs to restart or dependencies need to be installed.

## AI Features Fail

Check the server-only API keys validated in `apps/web/client/src/env.ts`. Missing keys should fail validation instead of being read from client code.

## Generated Preload Script

`apps/web/client/public/onlook-preload-script.js` is generated output. Change preload source under `apps/web/preload/script` and rebuild it through the normal maintainer workflow.
