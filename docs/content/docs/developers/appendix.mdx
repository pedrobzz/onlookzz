# Appendix

## Web Client Environment

`apps/web/client/src/env.ts` is the source of truth for required environment variables.

```bash
NEXT_PUBLIC_CONVEX_URL=
MORPH_API_KEY=
RELACE_API_KEY=
FIRECRAWL_API_KEY=
EXA_API_KEY=
```

## Local Project Storage

Project source files live on disk under:

```txt
.onlook/sandboxes/<project-id>
```

Convex stores project metadata, frames, conversations, messages, settings, checkpoints, and uploads. Convex does not store source file contents or raw filesystem watcher events.

## Runtime Responsibilities

The local runtime owns filesystem access, project watchers, install/dev/command processes, logs, preview ports, and runtime SSE events.
