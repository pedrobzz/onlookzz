# External Services

Onlook's core app data is handled by Convex and the local runtime. External services are only needed for optional AI/search capabilities.

## AI and Search Providers

Configure the server-only keys required by the features you enable:

```bash
MORPH_API_KEY=
RELACE_API_KEY=
FIRECRAWL_API_KEY=
EXA_API_KEY=
```

Add new provider settings through `apps/web/client/src/env.ts` so validation fails early when required configuration is missing.

## Runtime Access

Do not replace local runtime filesystem access with remote storage. Project source files must remain on local disk under `.onlook/sandboxes/<project-id>`.
